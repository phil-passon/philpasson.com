# Portfolio Website

A multilingual personal portfolio website built with HTML, CSS, and vanilla JavaScript.

## Features
* **Dual Language Support**: Integrated toggle system for German and English content using data attributes.
* **Theme Toggle**: Dark and light mode functionality with CSS variables and local state switching.
* **Tech Stack Grid**: Visual representation of skills using DevIcon integration.
* **Responsive Design**: Mobile-friendly layout using CSS Grid and Flexbox.

## Technical Details
* **Frontend**: HTML5, CSS3 (Custom Properties), JavaScript (ES6+).
* **Icons**: DevIcon SVG integration.
* **Deployment**: Configured for GitHub Pages compatibility.

## Structure
* `index.html`: Main structure and content containers.
* `style.css`: Theme variables and responsive layout definitions.
* `script.js`: Logic for language translation and theme switching.
